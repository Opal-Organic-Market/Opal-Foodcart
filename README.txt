# FoodCart - Agro Products E-commerce Website

## Overview

FoodCart is an e-commerce website specializing in agricultural products. This project was developed by a team of four individuals as part of the 12-week Generations MEST course. The website allows users to explore and purchase a variety of agro products, including fruits, vegetables, spices, poultry, seafood, and more.

## Features

1. **Product Listing:** Display a wide range of agricultural products with details such as name, seller, price, category, and an option to add items to the cart.

2. **Category Filter:** Users can filter products by category, making it easier to find specific types of agro products.

3. **Search Functionality:** The website features a search bar with suggestive search results, enhancing the user experience by providing relevant product suggestions as users type.

4. **Shopping Cart:** A simple shopping cart system allows users to add products to their cart for later purchase.

5. **Responsive Design:** The website is designed to be responsive, ensuring a seamless experience across various devices and screen sizes.

## Technologies Used

- HTML
- CSS (including MDB framework)
- JavaScript

## Getting Started

To run the website locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the `index.html` file in your preferred web browser.

3. Explore the website, filter products, perform searches, and experience the seamless e-commerce flow.
